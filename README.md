# 海滨体育馆管理系统 java186

## 项目概述

海滨体育馆管理系统是一个基于B/S架构的MVC模式应用，采用Java、SpringBoot、Mybatis、Vue等技术开发，旨在为体育馆管理人员和用户提供便捷的服务。

## 技术栈

- **架构**：B/S、MVC
- **系统环境**：Windows/Mac
- **开发环境**：
  - IDEA
  - JDK1.8
  - Maven
  - Mysql
- **技术栈**：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 管理员模块

- 个人中心
- 学生管理
- 器材管理
  - 器材借出管理
  - 器材归还管理
  - 器材分类管理
- 校队签到管理
- 进入登记管理
- 离开登记管理
- 活动预约管理
- 灯光保修管理
- 体育论坛
- 系统管理

### 学生模块

- 首页
- 器材
- 体育论坛
- 体育资讯

## 系统角色

- 管理员：负责管理体育馆各项事务。
- 学生：使用系统进行器材借用、活动预约等。

## 运行环境

- **服务器**：建议配置Tomcat服务器运行后端。
- **数据库**：Mysql数据库。
- **前端**：现代浏览器。

## 目录结构

```
|-- backend                # 后端代码
|   |-- src
|   |   |-- main
|   |   |   |-- java
|   |   |   |-- resources
|-- frontend               # 前端代码
|   |-- src
|   |   |-- assets
|   |   |-- components
|   |   |-- views
```

## 部署步骤

1. 准备好Mysql数据库环境。
2. 导入数据库文件到Mysql数据库中。
3. 部署后端代码到配置好的服务器上。
4. 部署前端代码到Web服务器目录。
5. 配置服务器安全策略，确保应用安全运行。

## 核心亮点

- **前后端分离**：基于SpringBoot和Vue的前后端分离架构，易于维护和扩展。
- **模块化设计**：功能模块划分清晰，便于未来的功能迭代和升级。
- **权限管理**：系统内置角色权限管理，确保数据安全。
